# 📊 Analyzer - NumPy Data Analytics Toolkit

## 📌 Description
Analyzer is a Python-based data analytics application built using NumPy and Object-Oriented Programming (OOP).

It allows users to create and manipulate arrays, perform mathematical operations, analyze data, and compute statistical insights through a menu-driven interface.

---

## 🚀 Features
- 📦 Create 1D, 2D, and 3D NumPy arrays
- ➕ Perform mathematical operations (add, subtract, multiply, divide)
- 🔗 Combine and split arrays
- 🔍 Search values in arrays
- 🔃 Sort and filter arrays
- 📊 Statistical analysis (mean, median, variance, std deviation)
- 📈 Advanced metrics (percentile, correlation)

---

## 🛠️ Technologies Used
- Python
- NumPy
- OOP (Classes & Methods)
